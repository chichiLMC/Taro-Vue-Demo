# Taro-Vue-Demo

## 基于Taro多端统一开发解决方案

## setup
```
npm install / yarn
```

### development
```
npm run dev:h5  / yarn dev:h5
npm run dev:weapp  / yarn dev:weapp
```

### production
```
npm run build:h5  / yarn build:h5
npm run build:weapp  / yarn build:weapp
```

## Listening
```
h5: http://0.0.0.0:10086/

miniProgram: 小程序工具导入 dist 文件夹
```
